NBA Player Salary Predictions

Machine learning models for predicting NBA player salaries

Project Overview

This project uses machine learning algorithms to predict NBA player salaries based on performance metrics. By analyzing historical data on player statistics, team success, and market conditions, I've developed models that can estimate player value and predict salary trends.

Data Sources & Features

The model incorporates multiple data sources and features:

  • Traditional statistics (points, rebounds, assists, etc.)
  • Advanced metrics (PER, WS/48, BPM, etc.)
  • Team performance and market size
  • Player age and experience
  • Position and role on team
  • Salary cap constraints

Modeling Approach

I employed several machine learning techniques including:

  • Random Forest Regression for feature importance analysis
  • Gradient Boosting for improved prediction accuracy
  • Linear regression with regularization for interpretability
  • Time series analysis for salary trend prediction

Key Insights

The analysis revealed several interesting patterns in NBA salary determination:

  • Advanced metrics like PER and WS/48 have higher predictive power than traditional stats
  • Team success and market size significantly impact player salaries
  • Age shows a clear peak around 27-28 years old
  • Position scarcity affects salary premiums

Applications

This work demonstrates the intersection of sports analytics and business intelligence. The models can be used by teams for contract negotiations, by agents for client valuation, and by analysts for understanding market dynamics in professional sports.
